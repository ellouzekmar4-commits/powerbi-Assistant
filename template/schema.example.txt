EXEMPLE REEL (base "csys" de la clinique) — pour inspiration :

Schema :
- dbo.Client : CodCli, Nom, CodRep (-> Represt.CodRep), codSouReg (-> soureg.codsoureg)
- dbo.Facture : numbon, codcli (-> Client.CodCli), codrep (-> Represt.CodRep), datesys
- dbo.MvtSto : numbon (-> Facture.numbon), codart (-> Stock.codart), desart, quantite,
  priuni, priach, montht, typmvt ('S'=vente, 'E'=retour, 'T'=transfert, 'X'=devis)
- dbo.Stock : codart, desart, FamArt (-> FamArt.FamArt)
- dbo.FamArt : FamArt, DesFam
- dbo.Represt : CodRep, Nom

Definitions :
- Chiffre d'affaires = SUM(montht) ou typmvt='S'
- Ventes Nettes = CA - retours (typmvt='E')
- Marge = SUM(quantite * (priuni - priach)) ou typmvt='S'

Jointures :
- CA par famille : MvtSto -> Stock -> FamArt (NE PAS joindre Client)
- CA par client : MvtSto -> Facture -> Client, avec filtre Nom non vide
